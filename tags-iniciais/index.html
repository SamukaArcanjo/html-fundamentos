Listas

Listas não Ordenadas

para listas não ordenadas você usa a tag ul, e adiciona a tag li para cada item dentro de ul

  • item 1

Listas Ordenadas

Para listas ordenadas você deve usar a tag ol no lugar da tag ul. O li continua da mesma forma

  1. item 1
  2. item 2
  3. item 3
