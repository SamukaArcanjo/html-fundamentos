Meu github

Você deve ter notado que adicionei o rel=noreferrer. O rel é um atributo usado para descrever a relação entre a página atual e o documento vinculado e pode assumir vários valores. Os mais relevantes são - Noopener Impede que a nova aba acesse a página original,o que abriria a possibilidade de ataques de phishing como o tabnabbing. Os navegadores modernos definem isso automaticamente para qualquer link que contenha link e target=_blank, mas você ainda verá essa configuração definida manualmente para manter a compatibilidade com navegadores antigos. - Noreferrer Semelhante ao anterior Noopener, mas tambpem impede que certos detalhes sobre a página original sejam passados para a nova página.
